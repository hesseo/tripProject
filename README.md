## 3. 프로젝트 소개

  - 여행을 좋아하는 사람들이 여행 정보를 공유하고, 반응을 남길 수 있습니다.
  - 여행을 함께할 멤버를 모집하는 글을 올리고, 채팅 기능을 통해 서로 소통할 수 있습니다.
  - 원하는 목적지의 교통편 길찾기를 제공할 수 있습니다. 
  - 지도 기능을 통해 장소 정보를 얻고 리뷰, 별점, 찜 기능을 할 수 있습니다.
  - 자신의 여행 일정을 타임라인을 통해 정리할 수 있습니다.
